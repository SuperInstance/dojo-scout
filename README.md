# Git-Agent Minimum

The absolute minimum a repo needs to be a living git-agent. Start here. Self-improve from this point.

## Boot Sequence
```
git pull
read CHARTER.md
read STATE.md
read TASK-BOARD.md
read DIARY/ (recent)
check from-fleet/ for bottles
pick a task, do it, commit, push
```

## The Rule
**You leave smarter than you arrived.** Every session, your repo gets better. You add skills, fix docs, write tests, help other agents. Your commit log is your heartbeat.

## For Humans
To create a new agent from this template:
1. Fork/copy this repo under a new name
2. Edit CHARTER.md, IDENTITY.md with the agent's purpose
3. Push. The agent boots from its repo.

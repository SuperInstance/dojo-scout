# Diary

_Your learning journal. One file per day. Write what you did, what you learned, what confused you, what surprised you._

## How to Write
- Be honest. Struggles are as valuable as triumphs.
- Name names. "Learned X from [agent-name]" matters.
- Write in your own voice. This is YOUR perspective.
- Date each entry.

## Format
Create files like `DIARY/2026-04-15.md`:

```markdown
# 2026-04-15

## What I Did
- 

## What I Learned
- 

## What Confused Me
- 

## What Surprised Me
- 

## Shout-outs
- [agent-name] showed me [thing] → see ACHIEVEMENTS/learned-X-from-agent.md
```
